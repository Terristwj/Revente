# SMOOVE

An Uber clone for SMU. PWA using Vue, TailwindCSS, NodeJS, Google Maps

## Tech Stack

-   frontend npm libraries
    -   axios
    -   lodash
    -   pinia
    -   pinia-plugin-persistedstate
    -   vue-material-design-icons
    -   vue-pwa-asset-generator
-   backend npm libraries
    -   axios
    -   body-parser
    -   cors
    -   dotenv
    -   express
    -   nodemon
